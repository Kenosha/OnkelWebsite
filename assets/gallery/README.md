# Galerie-Bilder

Lege deine Bilder in die passenden Ordner und benenne sie nummeriert:

- Dateinamen: `01` bis `12` (z.B. `01.jpg`, `02.webp`, `03.png`, …)
- Unterstützte Endungen (Priorität): `avif`, `webp`, `jpg`, `jpeg`, `png`, `svg`

Ordner:
- `assets/gallery/portrait-uhren/personalisierbar/`
- `assets/gallery/portrait-uhren/leuchtfarben/`
- `assets/gallery/paradoxe-gebilde/uhren/`
- `assets/gallery/paradoxe-gebilde/kerzenhalter/`
- `assets/gallery/paradoxe-gebilde/deko/`

Hinweis: In jedem Ordner liegen Platzhalter-`*.svg`. Sobald du echte Bilder mit gleicher Nummer ablegst (z.B. `01.jpg`), werden diese automatisch bevorzugt.
